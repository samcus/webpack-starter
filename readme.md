# Installation
```sh
$git clone https://github.com/samcus/webpack-starter.git
$npm i -g webpack
$npm install
```

# Run
```sh
$webpack --watch
#or
$npm run watch
#or with server + live reload
$npm run server
```

# Includes
- TypeScript Loader
- JQuery
- MomentJS
- SCSS Compilation
- Real Stylesheets Fallback (When JS is Disabled on the Client)
- Dev Server with Live Relead
- Desktop Notifications
