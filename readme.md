# Installation
```sh
$npm i -g webpack
$npm install
```

# Run
```sh
$webpack --watch
#or
$npm run watch
#or with server + live reload
$npm run server
```

# Includes
- JQuery
- MomentJS
- SCSS Compilation
- Dev Server with Live Relead
