# Installation
```sh
$npm i -g webpack
$npm install
```

# Run
```sh
$webpack --watch
#or
$npm run watch
#or with server + live reload
$npm run server
```

# Includes
- JQuery
- MomentJS
- SCSS Compilation
- Real Stylesheets Fallback (When JS is Disabled on the Client)
- Babel Transpilation
- Dev Server with Live Relead
